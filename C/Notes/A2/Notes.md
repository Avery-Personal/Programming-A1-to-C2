## Concepts Learnt

**Variables & Types**
`char` - Character/Single Character
`string` - String/Array of Characters

`int` - Integer/Whole Numbers
`float` - Float/Decimal Numbers

**Input & Output**
`printf` - Print text and values
`scanf` - Read user input

Format Values:
  %d - Integer Placeholder
  %i - Integer
  %f - Float

  %c - Character
  %s - String

**Arithmetic & Operators**
Basic Math: `+ - * /`
Compound Operators: `+=`, `-=`, `*=`, `/=`
